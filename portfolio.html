Pixel Art Maker

I've made this on Udacity, where I've received a scholarship for Front-End Web Dev. This web page application allows the users to draw pixel art on a customizable canvas. It’s pretty simple and fun. The user chooses the height, width and color. The color can be easily chosen. After all, the user hit “Send” and the grid is generated. Also, to use the color, it has to click inside any square.

It starts with the function ‘makeGrid()’; - inside the function we have defined the variables and the two ‘for’ loops; - to choose the height and width, I use a ‘for’ loop and inside each one I appended (defines a row in an HTML table) and (defines a standard cell in an HTML table) - after finishing the function it follows the event listener for coloring the cells - in the end of the code we have the event that calls the function at ‘submit’, and the grid is generated

Office building (metal structure) - 10 levels (Ds+P+8E)

This is my bachelor degree project. And it is composed of two parts: structure and management, as it follows:
The structure part:

    Written part:
  • Technical report
  • Evaluate uploads
  • Modeling the bracing frame
  • Diagonal dimensioning
  • Enlargement factor evaluation
  • Sizing upright current areas and pillars
  • Dimensioning base pillars
  • Assembly joining
  • Dimensioning floor girder.
    • Planes:
  • Marginal pillar + execution details
  • Current girder (Antiseismic Contour Frame) + details
  • Diagonal (Antiseismic Frame) + details
  • Floor plan + details
  • Details bolt anchor + details of it
  • Details secondary and principal girder (Gravitational Frame)

The part of management and economy:

    Written part:
  • Measurement
  • General estimate
  • Find offer
  • Analytically
  • Lists of works
  • Lists of resource consumption: materials, human, machinery and transport
  • Establishing execution speeds and workgroups
  • Schedule work execution through the chain method, including the Gantt chart type chart and human resources graph
  • Planning work execution through the MPM process, including the Gantt chart type chart and human resources graph
  • Establishment of the yard population and its structure; Social-administrative facilities related to the organization of the site
  • Occupational health and safety management system
    • Planes:
