Pixel Art Maker

I've made this on Udacity, where I've received a scholarship for Front-End Web Dev. This web page application allows the users to draw pixel art on a customizable canvas. It’s pretty simple and fun. The user chooses the height, width and color. The color can be easily chosen. After all, the user hit “Send” and the grid is generated. Also, to use the color, it has to click inside any square.

It starts with the function ‘makeGrid()’; - inside the function we have defined the variables and the two ‘for’ loops; - to choose the height and width, I use a ‘for’ loop and inside each one I appended (defines a row in an HTML table) and (defines a standard cell in an HTML table) - after finishing the function it follows the event listener for coloring the cells - in the end of the code we have the event that calls the function at ‘submit’, and the grid is generated
